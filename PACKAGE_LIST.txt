## 打包命令示例

### Windows PowerShell
```powershell
# 创建部署包目录
New-Item -ItemType Directory -Force -Path "deploy_package"

# 复制必需文件
Copy-Item config.py deploy_package\
Copy-Item main.py deploy_package\
Copy-Item index.html deploy_package\
Copy-Item requirements.txt deploy_package\
Copy-Item logo.jpeg deploy_package\
Copy-Item DEPLOY.md deploy_package\
Copy-Item PACKAGE_LIST.txt deploy_package\

# 创建空目录
New-Item -ItemType Directory -Force -Path "deploy_package\temp_audio"
New-Item -ItemType Directory -Force -Path "deploy_package\exports"

# 压缩为zip
Compress-Archive -Path deploy_package\* -DestinationPath voice-system-deploy.zip -Force
```

### Linux/Mac Terminal
```bash
# 创建部署包目录
mkdir -p deploy_package/temp_audio deploy_package/exports

# 复制必需文件
cp config.py main.py index.html requirements.txt logo.jpeg DEPLOY.md PACKAGE_LIST.txt deploy_package/

# 压缩
tar -czvf voice-system-deploy.tar.gz deploy_package/
```

## 上传到服务器的命令

### 使用scp (Windows PowerShell / Linux / Mac)
```bash
# 上传压缩包到服务器
scp voice-system-deploy.zip user@your-server-ip:/home/user/

# 登录服务器解压
ssh user@your-server-ip
cd /home/user
unzip voice-system-deploy.zip
mv deploy_package 03_Project_video
cd 03_Project_video
```

## 注意事项

1. **配置文件修改**: 部署前必须在config.py中修改大模型地址
2. **权限设置**: 确保exports和temp_audio目录有写入权限
3. **防火墙**: 确保服务器防火墙允许8000端口访问
4. **网络**: 确保服务器能够访问内网大模型API
